# Social Network Influence Maximizer

This project investigates the maximization of influence within social networks using AI-driven simulation models.

## Project Scenarios
1. **Personal Network:** A simulation of a small friendship group to identify the most central influencer (Luma, Ghasaq, Rawaa, Zainab).
2. **Social Media Simulation:** A directed network model showing how content spreads from a source to multiple subscribers.

## Technical Details
- **Language:** Python
- **Library:** NetworkX
- **Core Model:** Independent Cascade Model (ICM)
